# pizzaFinal

App created with React Node Express and Auth0


## Get Started

Install the client project dependencies:

```bash
npm init
```

```bash
cd client && npm init
```

Run the client project:

```bash
npm start
```

Run the server project:

```bash
cd server/server.js && node server.js
```

### Dependencies

@auth0/auth0-react
react
react-bootstrap
react-dom
react-router-dom
react-scripts
react-syntax-highlighter
